Recomendador de Peliculas

¿Para qué sirve?

El usuario medio toma de media 13 minutos para decidirse por que tipo de contenido ver en servicios de streaming, como Netflix

Con este proyecto, se te da ayuda a la hora de elegir que película ver o saber si merece la pena alguna que te genere dudas

¿Por qué usar Big Data?

  1. Es necesario procesar grandes cantidades de datos en el menor tiempo posible
  2. Cuantos más datos tengamos, mayores posibilidades se le pueden ofrecer al usuario
  3. Necesitamos dar con una solución lo antes posible

Funcionalidades

  • Recomendar peliculas en base a un genero determinado.
  • Recomendar peliculas según otra pelicula.
  • Recomendar peliculas según un año o región.
  • En estos 3 casos, se dará la opción de indicar el numero de peliculas que mostrar

  • Decidir si una pelicula merece la pena ser vista o no.
  • De forma adicional. indicar si una pelicula es adecuada para menores o solo adultos.
  • Descargaloaquí.
  • Cada vez que se quiera usar el proyecto, sólo hará falta ejecutar el archivo "main.py".
  • Si es la primera vez que se usa el software o se deseean actualizar los datos, habrá que ejecutar la opcion 0 del menú.

Rendimiento

  • Como se puede observar, de forma local baja el tiempo de ejecución según aumenta el número de cores.
  • Sin embargo, en la nube, el tiempo es más alto debido a la hora de cargar los datos.
